A Nu / Hkongso Keyboard

Keyboard Layout to allow typing of tones and ü vowel

This keyboard provides a smart keyboard layout for typing a Nu and Hkongsu. It should work with Unicode fonts such as Doulos SIL.

This Keyboard requires either Tavultesoft Keyman (Windows), Ekaya (Windows) or SCIM with KMFL support (Linux). You can then click on the keyboard icon on the task bar and select the Keyboard . To type again in English, click on the Keyman icon again and select “English” or “No Keyman Keyboard”.

Keyboard Layout

The keyboard layout is the same as for US, except that tone marks can be added after some vowels. Marks shown in white are typed by pressing the specified key after a vowel. Marks shown in green are typed by pressing Shift + the specified key after a vowel.
